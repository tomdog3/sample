# 開発内容
顧客管理（本人確認書類等認証機能の模倣）

# 要件
- ユーザが申し込みできること
- 本人確認書類のアップロードができること
- 管理者がユーザ情報の閲覧できること

# 機能
### ●管理者
- ログイン
- 顧客情報一覧表示
- 顧客情報編集・削除

### ●ユーザ
- ユーザ情報登録
- ファイルアップロード
- メール受信

# 開発環境
- MacOS Sierra 10.12.6
- NetBeans 8.2
- PHP 5.4
- FuelPHP 1.8
- MAMP 4.2.1

# モジュールパス
APPPATH=fuel/app/

### ●管理者
APPPATH/modules/admin

### ●ユーザ
APPPATH/modules/user
